# BIM Developer Masterclass Landing Page

## Overview

High-conversion landing page with distinctive "Dark IDE" aesthetic for BIM Developer Masterclass course. Targets architects, civil engineers, and BIM coordinators transitioning to developer roles.

**Tech Stack:** Next.js 16 + React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui
**Design:** VS Code-inspired dark theme with syntax-highlighting accents
**Component Strategy:** Extend shadcn/ui components with IDE aesthetic via CSS variables
**Goal:** Professional developer tool aesthetic with conversion optimization
